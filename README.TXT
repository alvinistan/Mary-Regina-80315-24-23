Centricity DICOM Viewer
========================================================================================

System Requirements
-------------------

The following requirements are necessary to effectively run the Centricity DICOM Viewer.
• Windows NT2000 SP4 or later, Windows XP SP2 desktop or laptop
• Internet Explorer 6.0 SP1, SP2 or later
• Minimum Pentium III processor 700 MHz
• Minimum 128 MB memory

If Internet Explorer is not your default browser the Centricity DICOM Viewer may not
start. If this is the case, follow the steps below.
1. Open Internet Explorer.
2. From the Tools menu across the top, select Internet Options.
3. Click on the Programs tab in the Internet Options window.
4. Click the Reset Web Settings button.
5. Click Yes on the Reset Web Settings pop-up.
• If you do not want to reset your home page, click on the box next to that message to
remove the check and disable the command.
6. Close Internet Explorer and restart the Centricity DICOM Viewer.

Recommended Minimum Configuration
---------------------------------

• Pentium 4 1 GHz
• 256 MB memory
